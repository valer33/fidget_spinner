# App Icon

Place your app icon here as `app_icon.png`

## Requirements
- Minimum size: 1024x1024 pixels
- Format: PNG (with alpha channel for transparency)
- Name: must be exactly `app_icon.png`

## How to Generate Icons

1. Create or find a 1024x1024 app icon image
2. Save it as `app_icon.png` in this folder
3. Run: `flutter pub run flutter_launcher_icons`

This will generate all the necessary icon sizes for:
- iOS (AppIcon.appiconset)
- Android (mipmap-* folders)
- Web (icons folder)

## Design Suggestion
For the FormFresh Fidget app, use:
- Cyan (#00D4FF) as the primary color
- Clean, minimalist spinner design
- Dark background to match the app aesthetic
